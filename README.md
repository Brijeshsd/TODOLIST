# 📝 To-Do List App

A simple, clean, and responsive To-Do List web app built using **HTML**, **CSS**, and **JavaScript**.

## 🔧 Features

- Add new tasks
- Remove tasks by clicking on them
- Auto-clear input after adding
- Responsive design
- Simple UI/UX

## 💻 Tech Stack

- HTML
- CSS
- JavaScript (Vanilla)
